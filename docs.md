addition

buttonPress - Обнаруживает нажатия кнопок на пинах А0, А1, А2. Принимает номер кнопки
secFromStart - Секунды со старта платы
minFromStart - Минуты со старта платы
hourFromStart - Часы со старта платы
startTime - Время, отсчитывается со старта платы. Принимает как дополнительный аргумент секунды для добавления к отсчёту времени
init - Инициализация Serial, randomSeed, встроенного светодиода, проверка EEPROM и сигнализация об ошибках. Использует пин А7 по умолчанию для генерации randomSeed, если он занят, вам нужно заменить его на другой плавающий пин
startTimer - Запускает миллисекундомер, полезно для измерения времени работы функции
getTimer - Возвращает пройденные миллисекунды
endTimer - Останавливает секундомер и обнуляет его

Settings

Settings - Инициализатор, можно добавить размер EEPROM для использования в других платах
resetAll - Затереть все значения в памяти нулями. Можно выбрать другое значение
save - Сохранить значение по адресу addr
saveBool - Сохранить логическое значение по адресу addr
load - Прочитать байт с EEPROM
loadVar - Прочитать байт по адресу в переменную
loadClamped - Прочитать байт с EEPROM, но только в области, задаваемой параметрами

TwoColor

control - Управление двухцветным светодиодом. Использует константы вида: Off, Green, Red
setRed - Задать двухцветному светодиоду красный цвет. Инвертируется
setGreen - Задать двухцветному светодиоду зелёный цвет. Инвертируется
disable - Выключить двухцветный светодиод
blink - Мигать двухцветным светодиодом с указанными цветом, временем
pinSet - Задать пины для двухцветного светодиода. По умолчанию 3 и 6
invert - Инвертировать двухцветный светодиод
